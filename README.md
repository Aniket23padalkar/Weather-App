# Weather App

## Live Link

[Weather App](https://aniket23padalkar.github.io/Weather-App/)

## Description

This is my **second API-based project**. I built a weather app that lets you search any city and see real-time weather data. I used the **OpenWeatherMap API** to fetch data and update it dynamically on the page.

## What I Learned

-   How to **fetch data from APIs** and work with JSON.
-   How to **display API data dynamically** in the DOM.
-   Error handling when the city name is wrong.
-   Using **template literals** to inject API results easily.
-   Learned how to show/hide error messages, default message, and results properly.

## Tech Stack

-   HTML
-   CSS
-   JavaScript (Vanilla)
-   OpenWeatherMap API
-   FlagsAPI (for country flags)

## Features

-   Search weather by city name.
-   Shows:
    -   Country flag & location
    -   Weather icon & description
    -   Temperature in °C
    -   Clouds, Humidity, Pressure
-   Error handling for wrong city names.
-   Default message on start.
